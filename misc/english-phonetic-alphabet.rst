English Phonetic Alphabet
=================================


Vowels
---------------------

::

    [I] [ə] [ɒ|ɑ] [u] [Λ] [æ] [e|ɛ]

    [i:] [ə:|ɜː] [ɔ:] [u:] [ɑ:]

    [aI] [eI] [ɔI] [au|oʊ] [əu] [Iə] [eə] [uə]

    
    Short vowels

    | IPA               | Sound                           | Example          |
    | ----------------- | ------------------------------- | ---------------- |
    | /ɪ/               | near-close near-front unrounded | *sit*            |
    | /e/ or /ɛ/        | open-mid front unrounded        | *bed*            |
    | /æ/               | near-open front                 | *cat*            |
    | /ʌ/               | open-mid back unrounded         | *cup*            |
    | /ʊ/               | near-close near-back rounded    | *put*            |
    | /ɒ/ (RP) /ɑ/ (GA) | open back unrounded             | *cot* / *father* |


    Long vowels / Tense vowels

    | IPA  | Sound                 | Example   |
    | ---- | --------------------- | --------- |
    | /iː/ | close front unrounded | *see*     |
    | /uː/ | close back rounded    | *food*    |
    | /ɔː/ | open-mid back rounded | *thought* |
    | /ɑː/ | open back unrounded   | *father*  |
    | /ɜː/ | mid central unrounded | *bird*    |


    Diphthongs

    | IPA       | Sound    | Example |
    | --------- | -------- | ------- |
    | /eɪ/      | *face*   | *day*   |
    | /aɪ/      | *price*  | *my*    |
    | /ɔɪ/      | *choice* | *boy*   |
    | /aʊ/      | *mouth*  | *now*   |
    | /oʊ/      | *goat*   | *go*    |
    | /ɪə/ (RP) | *near*   | *here*  |
    | /eə/ (RP) | *square* | *care*  |
    | /ʊə/ (RP) | *cure*   | *tour*  |


Consonants
----------------------

::

    [p] [t] [k] [f] [θ] [s] [tr] [ts] [∫] [t∫]

    [b] [d] [g] [v] [ð] [z] [dr] [dz] [ʒ] [dʒ]

    [h] [m ] [n] [ŋ] [l] [r] [w] [j]


    | IPA  | Sound                            | Example   |
    | ---- | -------------------------------- | --------- |
    | /p/  | voiceless bilabial stop          | *pat*     |
    | /b/  | voiced bilabial stop             | *bat*     |
    | /t/  | voiceless alveolar stop          | *tap*     |
    | /d/  | voiced alveolar stop             | *dog*     |
    | /k/  | voiceless velar stop             | *cat*     |
    | /g/  | voiced velar stop                | *go*      |
    | /tʃ/ | voiceless postalveolar affricate | *chat*    |
    | /dʒ/ | voiced postalveolar affricate    | *jam*     |
    | /f/  | voiceless labiodental fricative  | *fan*     |
    | /v/  | voiced labiodental fricative     | *van*     |
    | /θ/  | voiceless dental fricative       | *think*   |
    | /ð/  | voiced dental fricative          | *this*    |
    | /s/  | voiceless alveolar fricative     | *see*     |
    | /z/  | voiced alveolar fricative        | *zoo*     |
    | /ʃ/  | voiceless postalveolar fricative | *she*     |
    | /ʒ/  | voiced postalveolar fricative    | *measure* |
    | /h/  | voiceless glottal fricative      | *hat*     |
    | /m/  | bilabial nasal                   | *man*     |
    | /n/  | alveolar nasal                   | *no*      |
    | /ŋ/  | velar nasal                      | *sing*    |
    | /l/  | alveolar lateral                 | *let*     |
    | /r/  | alveolar/posterior approximant   | *red*     |
    | /j/  | palatal approximant              | *yes*     |
    | /w/  | labial-velar approximant         | *we*      |
